# **App Name**: AuraModa

## Core Features:

- Intelligent Fashion Showcase: Dynamic product catalog with real-time stock status and inventory indicators.
- Secure Midtrans Checkout: Integrated payment processing using Midtrans Snap for secure, multi-method transactions.
- Curated Visual Gallery: A filterable media hub with a dedicated lightbox modal for high-fidelity brand photography.
- Persistent Shopping Cart: Localized cart management that maintains selection state across page navigation without a database.
- Smart Validation System: Real-time sanitization tool for contact forms to prevent XSS and ensure valid Indonesian communication data.
- AI Style Curator: An AI-powered tool that uses reasoning to suggest accessory pairings based on items currently in the user's cart.
- Responsive Mobile Navigation: A fluid, mobile-first navigation menu with gesture-friendly overlay and transitions.

## Style Guidelines:

- Primary Color: Deep Lavender (#A78BFA) evoking luxury and fluid creativity.
- Background Color: Obsidian Purple (#13111A), a dark, heavily desaturated variant for a high-fashion, high-contrast experience.
- Accent Color: Arctic Indigo (#A5B4FC) to provide analogous contrast for CTAs and highlights.
- Headline Font: 'Space Grotesk' (Sans-serif) for a modern, tech-forward, and precise brand identity.
- Body Font: 'Inter' (Sans-serif) for neutral, machined objectivity and high readability at small sizes.
- Asymmetrical agency-style grids with deep container shadows (0 10px 30px rgba(0,0,0,0.2)) to create depth.
- Sophisticated entrance animations (fadeIn and translateY) applied to individual page segments for a polished, cinematic feel.
